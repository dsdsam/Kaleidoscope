Running Model Examples

Let's run some demo-projects to see how the developed models look like. The DSDS DSE has several pre-developed models that can be found under Examples menu. Please select one of the model names from the Examples menu dropdown list of models.

The model is drawn on the Design Space located on the left of this Help Pane. The model is auto scalable and you can see it enlarged when you either expand the DSDS DSE window or close this Help Pane. When model is created in Design Space you can fiddle with it by applying to it different menu operations.

You can apply to the model any modification you might want to try. Those modifications will exist at runtime only. When the same model name is selected from Examples menu again the new model is programmatically regenerated from the scratch. In order to preserve all the modifications you just have accomplished use Project -> Sase Project As menu item to save the modified model on your hard drive.
