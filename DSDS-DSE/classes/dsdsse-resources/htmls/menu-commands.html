Menu and Toolbar Commands

When the DDSSE is started the default Project named "Proj 000" is automatically created.

The New Proj, Open, Save, Save As, Erase Proj, Print, and Exit operations are used to operate with entire Project.

New Proj:
The New Proj currentOperation removes existing Project and creates new one with default name "Proj XXX", where XXX is the number of projects created in session. The default name of the Project can be changed when it is saved with currentOperation Save As.

Open:
The Open currentOperation opens "File/Open" dialog allowing you to find and load the Project from hard drive.

Save:
The Save currentOperation saves the Project on hard drive with its default name and extension "prj".

Save As:
The Save As currentOperation opens "File/Save" dialog allowing you to replace the Project default name and directory with your own.

Erase Proj:
The Erase Proj currentOperation is similar to New Proj one. It removes all the Models and data from current Project but does not change its name.

Print:
The Print currentOperation prints out the Project on the printer.

Exit:
The Exit currentOperation closes the DDSSE.
 
 

How to create Model.

The Model is created on the basis of Multi-Color Logical Net (MCLNet) formalism.

When the DDSSE is started the default Model named "Model 000" is automatically set.

Use New Model menu or toolbar button to set new Model each time when new model creation is started. The new default Model name "Model XXX", where XXX is the number of models created in project, is set.

Creating the model is straight forward and takes fore simple steps.

  • First, you create Cells - the elements which represent discrete variables of the system.
  • Then you create Junctions - the elements which represent rules that are governing the Cells state.
  • As the third step, you connect the Junctions to the Cells and the Cells to the Junctions with Arcs.
  • Finally, you set initial state of the Cells and Arcs and save the Project on your hard drive.
That is all.

When operating with Model:

  1. use left mouse button to indicate new elements location or to pick up existing one;

  2. use right mouse button to bring an element popup menu, to undo preceding currentOperation step or to quit the currentOperation.

How to Create Cells.

To create Cells push the menu or toolbar button "Create Cells" and just pick points on the Project Area of the screen by pressing left mouse button to locate them or right mouse button to remove created Cell or quit the currentOperation.

How to Create Junctions.

There are two ways to create Junction: on the bases of "fragment" and by placing on the screen one of four "oriented" junctions.

To create Junction on the bases of fragment.

  1. Pick one of the Junction input Cell, then pick its output Cell. The Cells get highlighted in magenta.

  2. Pick the desired location for Junction. Two Arcs connecting selected Cells with Junction will appear.

  3. To finish creating the rule create additional Arcs using "How to Create Arc" technique.

To create oriented Junctions use one of four "Create Junction" menu or toolbar button allowing to create Vertical, Horizontal, Slanted to the Right and Slanted to the Left Junctions. Push the button with one of appropriate for particular case orientation and pick points on the Project Area of the screen by pressing left mouse button to locate Junction or right mouse button to remove created one or quit the currentOperation. To finish creating the rule create additional Arcs using "How to Create Arc" technique.

How to Create Arc.

The arc is created in rubber bending mode and goes from its input node (a Cell or a Junction) to its output node (a Junction or a Cell) respectively. Pick the input node of the arc and draw the mouse in the desired direction. The arc line will follow the mouse. The shape of the arc may be different and you have choices here.

  1. you can create straight arc by picking up output node right after the input node was picked and then picking this node again to indicate that creating is finished.

  2. you may create three point arc by picking up output node right after the input node was picked and then continuing to move mouse apart from output Arc. The Arc connecting the nodes will be banding according to current mouse location. Third pick will finish the Arc creating and the Arc Knob will be placed at the last point picked.

  3. you may create multi-knot arc which will go through multiple knots on the screen before it reaches the arc output node. To make that move the mouse to a position where the knot should be located and press the left mouse button. When you see the knot created continue moving mouse and creating additional knots. When all the knots are created pick the output node and then select one of the knots as a place where the Arc Knob should be placed.
