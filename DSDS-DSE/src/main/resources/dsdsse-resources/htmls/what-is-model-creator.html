The Graphical Model Creator

The Graphical Model Creator is a collection of tools and wizard-like components that allow the user to create, store, retrieve, modify and again store the Project and the Model.

Graphical Model Creator is enabled when DSE is set to Development mode. User switches DSE to Development mode by selecting menu item Development -> Set Development Mode. Switching to Development mode enables Creation and Modification menus.

User communicate with Graphical Model Creator via its Creation and Modification menus. Selecting each menu item makes DSE to allow user to accomplish several corresponding operations one after another.

Available under Creation menu operations are: Create Property Nodes, Create Condition Nodes, Create Arcs, Create Simple Fragment.

Available under Modification menu operations are: Move Entire Model, Move Model Fragments, and Delete Model Elements.

Selected operation activates development processes that drives user through the chain of development steps being suggested by the operation. The step execution instructions are displayed on the "Operation" and "Step" sections of the Information Bar. The Information Bar is located below the Design Space Toolbar.

After completion of each operation user can either repeat currently selected operation again, e.g. to create one more Property Node on the screen or to select another menu item that will drive user through the steps of execution of another operation, e.g. creating several Condition Nodes or slightly moving the model on the screen to preferred location. To finish executed operation user clicks on currently selected menu item again.
